# Open Design Kit

Agent-oriented toolkit for **structured product design**: research, systems, UI patterns, motion, and handoff-ready specs.

## Contents

| Path | Purpose |
|------|---------|
| [`SKILL.md`](./SKILL.md) | Main agent instruction — six-phase design workflow |
| [`design-systems/_base/DESIGN.md`](./design-systems/_base/DESIGN.md) | Brand and design-token schema template |
| [`skills/landing-page/SKILL.md`](./skills/landing-page/SKILL.md) | Landing page composition and conversion patterns |
| [`skills/ui-components/SKILL.md`](./skills/ui-components/SKILL.md) | Reusable UI component guidance |
| [`references/animations.md`](./references/animations.md) | Animation and motion pattern library |

## Usage

1. Point your agent at the root `SKILL.md` for full-process design tasks.
2. For scoped work, load the matching skill under `skills/` only.
3. Fill `design-systems/_base/DESIGN.md` per product; extend with product-specific tokens and components.
