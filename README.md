Agent-oriented toolkit for **structured product design**: research, systems, UI patterns, motion, and handoff-ready specs.

## Usage

1. Read [`HOW-TO-USE.md`](./HOW-TO-USE.md) for setup and prompt recipes.
2. Point your agent at the root `SKILL.md` for full-process design tasks.
3. For scoped work, load the matching skill under `skills/` only.
4. Fill `design-systems/_base/DESIGN.md` per product; extend with product-specific tokens and components.
